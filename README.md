## Sobre o Projeto

Este projeto demonstra a aplicação de **HTML Semântico**, explorando as principais tags e conceitos de estruturação de páginas web. O objetivo é criar uma interface organizada e acessível, seguindo as melhores práticas de desenvolvimento front-end.

---

<table>
<tr>
<td>

### Formulários & Inputs
- `<form>` - Estrutura de formulários
- `<input>` - Diversos tipos de entrada
- `<checkbox>` - Seleção múltipla
- `<radio>` - Seleção única
- `<button>` - Botões e tipos
- `<select>` - Listas suspensas
- `<textarea>` - Áreas de texto

</td>
<td>

### Formatação & Estrutura
- Formatação de Textos (Parte 1 & 2)
- `<div>` e `<span>` - Containers
- `<fieldset>` - Agrupamento de campos
- `<embed>` - Conteúdo incorporado
- `<iframe>` - Frames inline
- 🎨 Teoria das Cores
